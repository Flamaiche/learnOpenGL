MENU PRINCIPALE :
ENTER          : valider
MAJ (SHIFT)     : monter
CTRL            : descendre

EN PARTIE :

DEPLACEMENT
Z               : avancer
S               : reculer
Q               : aller à gauche
D               : aller à droite
MAJ (SHIFT)     : monter
CTRL            : descendre

ROTATION CAMERA
Flèche gauche   : tourner la caméra à gauche
Flèche droite   : tourner la caméra à droite
Flèche haut     : regarder vers le haut
Flèche bas      : regarder vers le bas

ROLL DU MONDE
A               : faire tourner le monde à gauche
E               : faire tourner le monde à droite

MODE CAMERA
ESPACE          : activer/désactiver le mode orbite (caméra libre / orbite)

SOURIS
Déplacement     : faire tourner la caméra (si curseur capturé)
Bouton gauche   : tirer
CAPS LOCK       : activer/désactiver le curseur (libérer ou capturer la souris) (verr. maj)

GESTION DU JEU
ECHAP           : quitter le jeu

HUD / DEBUG
                : menu de debug
