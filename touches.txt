MENU PRINCIPAL :

ENTRÉE : Valider / commencer la partie (appui unique)
MAJ (SHIFT) : Monter dans le menu (tant que pressée)
CTRL : Descendre dans le menu (tant que pressée)
ÉCHAP : Quitter le jeu (appui unique)

EN PARTIE :

DÉPLACEMENT
Z : Avancer (tant que pressée)
S : Reculer (tant que pressée)
Q : Aller à gauche (tant que pressée)
D : Aller à droite (tant que pressée)
MAJ (SHIFT) : Monter (tant que pressée)
CTRL : Descendre (tant que pressée)

ROTATION DE LA CAMERA
Flèche gauche : Tourner la caméra à gauche (tant que pressée)
Flèche droite : Tourner la caméra à droite (tant que pressée)
Flèche haut : Regarder vers le haut (tant que pressée)
Flèche bas : Regarder vers le bas (tant que pressée)
Déplacement de la souris : Tourner la caméra (si curseur capturé)

ROLL DE LA CAMÉRA (ALT + Q/E/R)
ALT + Q : Faire tourner la caméra / monde à gauche (tant que pressée)
ALT + E : Faire tourner la caméra / monde à droite (tant que pressée)
ALT + R : Réinitialiser le roll (appui unique)

MODE CAMÉRA
ESPACE : Activer le mode orbite (tant que pressée)
Relâchement de ESPACE : Désactiver le mode orbite

TIR
Bouton gauche souris : Tirer (appui unique, cooldown 0,3 s)

CURSEUR
CAPS LOCK : Activer/désactiver le curseur (verrouillage/déverrouillage de la souris) (appui unique)

GESTION DU JEU
ÉCHAP : Mettre en pause ou quitter selon le contexte (appui unique)

HUD / DEBUG
(Pas encore défini) : Menu de debug / informations HUD
